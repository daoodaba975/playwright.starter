A basic starter project with Playwright for modern web apps end-to-end testing.

## Install dependencies

```bash
npm install
```

## Run test script

Headless mode :

```bash
npx playwright test
```

UI mode :

```bash
npx playwright test --ui
```

HTML Test Reports :

```bash
npx playwright show-report
```
